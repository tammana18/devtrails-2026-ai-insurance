🚀 AI-Powered Insurance Platform for Gig Workers

A smart parametric insurance system designed to protect gig workers (delivery partners) from income loss caused by external disruptions such as weather, pollution, and unexpected events.



📌 Problem

Gig workers (Zomato, Swiggy, etc.) often lose income due to:
- 🌧️ Heavy rain
- 🌫️ Pollution
- 🚫 Curfews and disruptions

There is currently no system that provides income protection for these unpredictable situations.



💡 Solution

We developed an AI-powered insurance platform that:
- Detects external disruptions
- Automatically triggers claims
- Prevents fraudulent claims
- Instantly processes payouts
- Provides analytics through a dashboard



⚙️ Features

🌧️ Automated Claim Trigger
- Users can simulate disruptions using “Simulate Rain 🌧️”
- System automatically initiates claim processing

🛡️ Fraud Detection
- Identifies invalid or fake claims
- Rejects fraudulent claims with clear reason

 💸 Instant Payout
- Approved claims receive ₹500 (simulated payout)
- Instant feedback shown to user

📊 Dashboard
- Policy status
- Earnings protected
- Total claims
- Fraud detection count
- AI-based prediction



 🧠 AI Logic

- Risk prediction based on disruption scenarios  
- Fraud detection rules  
- Claim validation mechanism  



🏗️ Tech Stack

Frontend: React.js  
Backend: Node.js, Express.js  
Other: Mock APIs, Simulated data  



 🔄 Workflow

Login → Policy Selection → Simulate Disruption → Claim Trigger → Fraud Check → Approved/Rejected → Payout → Dashboard Update
